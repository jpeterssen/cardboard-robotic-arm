# Cardboard Robotic Arm

A motorized robotic arm built from cardboard and servo motors,
controlled by Arduino via PlatformIO

## Goals
- 3 degrees of freedom (base, shoulder, elbow)
- Gripper capable of picking up a ping pong ball
- Programmed pickup and place sequences

## Hardware
- Arduino UNO R3
- SG90 servo motor
- Corrugated cardboard

## Software
- PlatformIO
- Arduino Framework
